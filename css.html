•Flexbox CSS Properties:

display: flex | inline-flex - Defines the element as a flex container.
flex-direction: row| row-reverse | column | column-reverse - Sets the main axis direction :
justify-content: Specifies how items are distributed along the main axis.
align-items: Specifies how items are aligned along the cross axis.
align-self: Allows individual items to override their container's alignment.
flex: Shorthand for flex-grow, flex-shrink, and flex-basis properties.
flex-grow: Specifies the ability of an item to grow.
flex-shrink: Specifies the ability of an item to shrink.
flex-basis: Specifies the initial size of a flex item.
flex-wrap: Controls whether the flex container is a single-line or multi-line layout.
align-content: Aligns lines within a multi-line flex container.
